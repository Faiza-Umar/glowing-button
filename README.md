# glowing-button

## Overview

The **Glowing Button** is a modern front-end project built using semantic HTML and CSS.

The goal of this project was to create a visually engaging button using clean structure and CSS hover effects, demonstrating foundational UI styling skills and attention to interactive design.

---

---

## Key Features

- Interactive hover-based glow effect  
- Clean and semantic HTML structure  
- Lightweight and fast-loading  
- Simple and maintainable CSS  
- Easy to customize styling  

---

## Technologies Used

- **HTML5** – Structured markup  
- **CSS3** – Styling and hover effects  

---

## Technical Implementation

- Implemented a glowing effect using layered `box-shadow`
- Applied `:hover` pseudo-class for interactive behavior
- Structured CSS for readability and maintainability
- Designed with reusability in mind
